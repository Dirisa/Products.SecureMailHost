Installing CMFMetadata

  First make sure all requirements (see below) are installed. Then
  install CMFMetadata in Zope's 'Products' directory and restart
  Zope. See the instructions below on how to integrate it with Silva
  or the CMF.

CMFMetadata Requirements

  Formulator (1.5.0)+

     It can be found here:

      "Formulator":http://www.zope.org/Members/infrae/Formulator

  Annotations 

  ProxyIndex 

Silva Installation

  The metadata system comes bundled with Silva 0.9.2 and is installed as
  part of a Silva installation.

CMF Installation

  Requires CMF 1.4 (not tested with 1.3)

    In Plone use the quickinstaller tool to install the Annotations 
    Tool, next delete portal_metadata tool, and then install CMFMetadata
 
    or it can be installed manually via

    Add an external method install_metadata::

       Module: CMFMetadata.Install
       Function: install

    and click on the test tab.
