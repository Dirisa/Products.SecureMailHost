DCWorkflowGraph

    DCWorkflowGraph is a DCWorkflow graphic viewer now. It uses Graphviz. I want to make it a graphic editor for DCWorkflow, just like what OpenFlowEditor does.

Require

    "Graphviz":http://www.graphviz.org
